# Bootcamp React.js - IGTI 2021
Aplicações desenvolvidas no Bootcamp do IGTI (Instituto de Gestão e Tecnologia da Informação)
#

### Projeto 02 - React Countries
Construção de um filtro de países utilizndo um array de objetos contendo informações de diversos países.
O usuário pode selecionar o país a partir de um input, bem como clicar em algum item para marcá-lo ou desmarcá-lo como visitado.
